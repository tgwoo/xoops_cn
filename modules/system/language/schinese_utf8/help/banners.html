帮助主题：广告

说明

广告就是出现在页面上方的宣传广告。他们可以是图片或Flash，例如预设的三则广告就有一个是Flash的广告。XOOPS有内置一个简单的广告管理系统，让您可以建立委托人（或广告主）的资料。您可以同时放上多个广告，只要使用者连到网站上，就会随机秀出其中一则广告，每连到一个新页面，就会随机载入广告内容。

当广告每呈现一次，就会在「展示」计数一次，这可以让您或委托人清楚的知道此广告呈现了多少次。一般来说，若有人委托，那么都是以「展示」次数来计算费用的。

请注意：假如您是打算用在商业上（例如贩售广告业务），那么有个重要设置您应该去设一下：「后台→偏好设置→系统设置→一般设置→您的 IP 位址」您应该填入您网站的 IP 位置，以避免自己在修改或浏览网站时算到展示次数中。若没有这样做，将会被认为是不老实的！
