帮助主题：偏好设置

说明

此偏好设置界面是让您用来设置整个网站的相关设置用的，它值得您花点时间来研究之，了解一下哪些设置对于网站的表现、安全有什么样的影响。

一般设置

这里的设置通常是最重要，也最常需要设置的项目。网站架好之初，您必须到此设置网站名称以及标语，您也需要到此选择一个预设主题以及要使用的后台管理界面。日后若您的网站需要进行维护，您也可以到此暂时关闭您的网站！在此您也可以设置每个模块的快取时间。

Meta及页尾

Meta标签主要是用来提供一些网站信息给查询引擎用的，这和网路优化有点关係，这些标签最后会被置于所有的网页表头中。

要了解的是这是提供给查询引擎作为建立索引用的，设得好的话，可能会备受重视，设不好也可能导致查询引擎完全忽略之，所以，小心使用Meta标签功能，以便让您的网站可以快速的被查询，这或许会对于网路查询排名的提升有所帮助。

请尽量提供精确且真实的信息，不要试图利用它来操控查询引擎以获得更好的排名，因为一旦被发现将有可能会被查询引擎降级甚至列入黑名单。

敏感词语检查

当有用者发表文章时，XOOPS可以自动侦测一些不良的用语，并用一些您自定义的字词将之替换掉。由于这样的功能会耗掉一些系统资源，有可能会导致网站变慢一点点，所以预设是关闭的。

要启用它，请于「启用敏感词语检查」选「是」即可。您需要在「敏感词语」中输入一个词语列表，并用「|」隔开，大小写不拘，并输入「敏感词语将被取代为：」中要取代的文字（预设为#OOPS#），如此，一旦侦测到不当词彙，便会用您设置的文字来取代之。

查询选项

查询工具对于在网站中寻找资料是相当重要的。XOOPS的查询是根据模块来分类的，其查询结果的呈现项目则依模块用途而有所不同，例如链接模块就呈现网站链接、新闻模块则呈现完整文章的链接...等。

「启用全站查询」预设是有启用的，假如关掉了，那么查询功能也就没作用了。 预设的「关键字最小长度」为2个字，若输入字数小于「关键字最小长度」设置的值，则会直接忽略之，不进行查询的动作。字数设太少将导致传回大量数据，字数设太多则很难下关键字，所以，请自行衡量囉！

电子邮件设置

此处用来设网站的发送方式。建议您用预设的「php mail()」即可，其他的如「sendmail 路径」或「SMTP 伺服器」都可以不用去填。

认证方式管理

其实一般只要使用「XOOPS资料」即可，其馀的您的可以直接忽视。

用 LDAP 或 Microsoft Active directory servers 的认证方式也已经被XOOPS支持，因为很多公司都有用LDAP或AD来建立自己的使用者数据库，所以，日后他们就可以直接用自己的数据库来作为认证的依据。

系统基本设置

此处您可以启用或关闭每个后台的管理功能，也可以设置后台设置，例如图像管理员一页要列出几个图像。您也可以在此处针对区块、评论或一般模块来选择您想要的预设编辑器。
