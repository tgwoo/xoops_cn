如何开始使用 XOOPS?

XOOPS 是一个用PHP搭配MySQL数据库所写的网页平台，它是以物件导向方式来开发，所以是一个可以建立小至个人网站大至大型社群网站、公司网站或入口网站的理想工具。为了更有效率的应用XOOPS，您需要安装一些模块以提供网站所需要的功能。在官网的 Module Repository 有许多不同的模块，例如日历、相册、新闻、论坛、天气...等各式各样的模块。

此帮助中心是提供各个管理功能的基本信息，不管什么时候您想了解目前您所在的功能页面到底有什么功能，您只要点击右上角的图示即可开启此说明。r
